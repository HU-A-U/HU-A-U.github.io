小记--关于mitmproxy的代理连接被重置的问题
2019-05-27|python

概述

在浏览器设置proxy的代理之后，访问网页显示连接被重置，然后查看日志，报错信息如下：

可以看到 ‘killed by block_global’ 连接被kill掉

解决

去官网找到解决办法

启动参数 添加 –set block_global=false 将block_global设为false

试了一下

-s 参数是使用启动脚本进行启动

又恢复正常，至于原因不是很清楚，因为之前一直运行很正常，可能是跟网络有关。
